A flying game developed using React, Three.js and GitHub Copilot(ChatGPT, Claude Haiku)

1. Cattles, Clouds, Watch towers are placed randomly.
2. Rain, Thunder, Audio effect.
3. Left, Right, Up, Down arrow controls.
4. Mouse wheel scroll (Zoom in, Zoom out) controls.
5. Hot air balloon is the flight avatar (3D model got from free3d site).
6. Infinite grass ground(wind visible on reaching lowest altitude).
